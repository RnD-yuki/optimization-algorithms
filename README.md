# Optimization Algorithms

最適化アルゴリズムおよび基本データ構造の実装集です。  
主にJavaを用いて実装していますが、アルゴリズムの性質に応じて他言語での実装を追加する場合があります。  

2024年に作成したコードをベースに、一部整理・再構成を行いながら公開しています。

## 実装内容
### Union-Find
- 経路圧縮を用いた互いに素な集合（Disjoint Set）の管理
-連結性判定やグラフ関連アルゴリズムに使用
### 貪欲法(区間スケジューリング）
- 重なりのない区間を最大数選択するアルゴリズム
- 終了時刻によるソートを基に構成

### 動的計画法（Longest Common Subsequence）
- 最長共通部分列（LCS）を求めるアルゴリズム
- DPテーブルの構築および解の復元を含む

## 目的
アルゴリズムの理解およびシステム設計への応用を目的として実装しました。
